# Data Immersion & Wrangling - Marketing Campaign Dataset
## Internship
ApexPlanet Private Limited
## Project Overview
This project focuses on data cleaning, preprocessing, and feature engineering using the Marketing Campaign dataset from Kaggle.
## Objective
The objective of this project is to prepare raw data for analysis by identifying and resolving data quality issues such as missing values, duplicate records, inconsistent formats, and outliers.
## Tools Used
* Python
* Pandas
* Visual Studio Code
## Dataset
Marketing Campaign Dataset (Kaggle)
## Tasks Performed
### 1. Data Loading
* Imported dataset using Pandas.
* Explored dataset structure and column information.
### 2. Data Quality Assessment
* Checked missing values.
* Identified duplicate records.
* Verified data types.
### 3. Data Cleaning
* Replaced missing values in the Income column using the median value.
* Removed duplicate records.
### 4. Data Transformation
* Converted Dt_Customer into datetime format.
### 5. Feature Engineering
Created new features:
* Age
* Total_Spent
* Total_Children
### 6. Outlier Detection
* Applied IQR method on Income column.
* Identified unusual income values.
### 7. Data Validation
* Confirmed removal of missing values.
* Confirmed removal of duplicate records.
## Output
Generated a cleaned dataset:
* cleaned_dataset.csv
